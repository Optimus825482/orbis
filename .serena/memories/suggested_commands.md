# Suggested Commands for ORBIS

## Development & Execution
- **Run the app:** `python run.py` or `flask run`
- **Install dependencies:** `pip install -r requirements.txt`
- **Environment setup:** Copy `env.example` to `.env` and fill in API keys.

## Testing (Currently none, proposed)
- **Run tests:** `pytest` (once implemented)
- **Check coverage:** `pytest --cov=.` (once implemented)

## Utility Commands
- **Check git status:** `git status`
- **List files:** `ls` or `dir` (Windows)
- **Search code:** `grep -r "pattern" .` (in Git Bash) or `findstr /s "pattern" *` (Windows CMD)
